Language Translator using Angular.js

In this experiment Angular.js is used to translate the text inputted by the user to 3 different languages. This experiment shows how we can perform view (html element) and the model (javascript) binding using Angular.js.

Demo:

Russian: {{ruText}}
French: {{frText}}
German: {{deText}}

Code snippet:

HTML: ... <div> <label for="enText">English: </label> <input name="enText" ng-model="enText" ng-change="translate()" text="car"/> </div> <div> <b>Russian: </b> {{ruText}} </div> <div> <b>French: </b> {{frText}} </div> <div> <b>German: </b> {{deText}} </div> Javascript: $scope.translate = function () { var input = $scope.enText; $http.jsonp("https://translate.yandex.net/api/v1.5/tr.json/translate?key=trnsl.1.1.20150214T214754Z.f24524a6ba947e55.0b517ffb397939ef326b4b298f5c1944e8de47c4&lang=en-ru&text="+input+"&callback=JSON_CALLBACK"). success(function (response) { $scope.ruText = response.text[0]; }); $http.jsonp("https://translate.yandex.net/api/v1.5/tr.json/translate?key=trnsl.1.1.20150214T214754Z.f24524a6ba947e55.0b517ffb397939ef326b4b298f5c1944e8de47c4&lang=en-fr&text=" + input + "&callback=JSON_CALLBACK"). success(function (response) { $scope.frText = response.text[0]; }); $http.jsonp("https://translate.yandex.net/api/v1.5/tr.json/translate?key=trnsl.1.1.20150214T214754Z.f24524a6ba947e55.0b517ffb397939ef326b4b298f5c1944e8de47c4&lang=en-de&text=" + input + "&callback=JSON_CALLBACK"). success(function (response) { $scope.deText = response.text[0]; }); } ...

On change of the value in the input box for english text string a function is invoked, which will make JSONP API calls and get Russian, French and German versions of the input. This translated text is assigned to the UI bound variables and hence it can be displayed on the screen immediately.
