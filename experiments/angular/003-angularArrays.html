Array Handling with Angular.js

This experiment shows how we can manipulate arrays using Angular.js. The experiment demonstrates a simple items check-list in which we can add new items, deleted previously added items and mark items as complete.

Code snippet:

HTML: ... <table class="table-condensed"> <thead> <th>&nbsp;</th> <th> Item </th> <th> &nbsp; </th> </thead> <tbody> <tr> <td>&nbsp;</td> <td> <input type="text" ng-model="newItem"/> </td> <td> <button type="button" class="btn btn-success" ng-click="addItem()"> <span class="glyphicon glyphicon-plus" aria-hidden="true"></span> </button> </td> </tr> <tr ng-repeat="item in items"> <td><input type="checkbox" ng-click="temp(item)" ng-checked="item.isComplete"</td> <td> <label class="done-{{item.isComplete}}">{{item.description}}</label> </td> <td> <button type="button" class="btn btn-danger" ng-click="removeItem(item)"> <span class="glyphicon glyphicon-remove" aria-hidden="true"></span> </button> </td> </tr> </tbody> </table> Javascript: var items = [ { "isComplete": false, "description": "Bread" }, { "isComplete": false, "description": "Butter" }, { "isComplete": false, "description": "Apples" }, { "isComplete": false, "description": "Juice" }, ]; $scope.items = items; $scope.temp = function (item) { item.isComplete = !item.isComplete; } $scope.addItem = function () { var newItem = $scope.newItem; $scope.items.push({ "isComplete": false, "description": newItem }); $scope.newItem = null; } $scope.removeItem = function (item) { items.splice(items.indexOf(item), 1); } ...

The two important methods, as shown in the snippet, to add and remove items from a list in Angular.js is "push" and "splice" respectively. Also, "ng-checked" can is used to toggle the check-box value and a css class selector along with the checkbox value is used to strike-through an item that is checked.
