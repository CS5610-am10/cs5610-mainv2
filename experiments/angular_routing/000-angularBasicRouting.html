Basic routing with AngularJS

This experiment is an introduction to routing with Angular.js and shows how we can use Angular.js to render multiple views without actually reloading the html page. Click on the 2 page links below and view different views without the page being reloaded.

Demo:

Page 1 Page 2

Code snippet:

HTML: ... <nav> <a href="#/page1">Page 1</a> <a href="#/page2">Page 2</a> </nav> <div ng-view> </div> Javascript: var app = angular.module('AngularBasicRoutingApp', ['ngRoute']); app.config(['$routeProvider', function ($routeProvider) { $routeProvider. when('/page1', { templateUrl: 'partials/page1.html' }). when('/page2', { templateUrl: 'partials/page2.html' }). otherwise({ redirectTo : '/page1' }); }]); ...

In order to use angular routing, the angular app module must depend on the 'ngRoute'module. Also, as shown in the above script, the route provider is configured to display different views (html) for different routes.
