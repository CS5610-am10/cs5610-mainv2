Bootstrap Responsive Utilities

This experiment shows how we can use bootstrap css to control the visibility of html elements based on the window screen size. Go ahead and change the width of the window.

Demo:

This paragraph is visible only on extra small devices.

This paragraph is visible only on small devices.

This paragraph is visible only on medium devices.

This paragraph is visible only on large devices.

Code Snippet:

... <p class="visible-xs-block">This paragraph is visible only on extra small devices.</p> <p class="visible-sm-block">This paragraph is visible only on small devices.</p> <p class="visible-md-block">This paragraph is visible only on medium devices.</p> <p class="visible-lg-block">This paragraph is visible only on large devices.</p> ...
  1. .visible-xs-*: for screen size less than 768px. It is hidden on other devices.
  2. .visible-sm-*: for screen size greater than or equal to 768px and less than 992px. It is hidden on other devices.
  3. .visible-md-*: for screen size greater than or equal to 992px and less than 1200px. It is hidden on other devices.
  4. .visible-lg-*: for screen size greater than or equal to 1200px. It is hidden on other devices.

Similarly we can use the "hidden-*-*" class to hide html elemets.
