Bootstrap Navigation Menu

This experiment shows how we can use bootstrap css, bootstrap java script and jquery to create a navigation bar for a website. If viewing on a large window reduce the size of the window to see how the menu items collapse into a menu drop-down

News
  • Weather
  • Finance
  • Entertainment

Code Snippet:

... <nav id="Nav1" class="navbar navbar-fixed-top navbar-inverse " role="navigation"> <!-- Brand and toggle get grouped for better mobile display --> <div class="container"> <div class="navbar-header"> <button type="button" class="navbar-toggle" data-toggle="collapse" data-target="#navbarCollapse"> <span class="sr-only">Toggle navigation</span> <span class="icon-bar"></span> <span class="icon-bar"></span> <span class="icon-bar"></span> </button> <a class="navbar-brand" href="hw4.htm" target="_blank">News</a> </div> <!-- Collect the nav links, forms, and other content for toggling --> <div class="collapse navbar-collapse" id="Div1"> <ul class="nav navbar-nav"> <li class="active"><a href="#" target="_blank">Weather</a></li> <li class="active"><a href="#" target="_blank">Finance</a></li> <li class="active"><a href="#" target="_blank">Entertainment</a></li> </ul> </div> </div> </nav> ...

As shown in the above code snippet we use various robust bootstrap classes to achieve a well defined navigation menu.
