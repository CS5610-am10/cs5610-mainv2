Bootstrap Grid Layout

This experiment shows how we can use bootstrap css to layout our content in a grid manner. In this experiment some content is showed in columns side by side for large devices. For small devices the content will be stacked one below the other.

Code Snippet:

... <div class="row"> <div class="col-lg-4"> <h2>Making use of col-xs-4</h2> <p>This is column 1</p> </div> <div class="col-lg-4"> <h2>Hi !! I am 2nd col-xs-4</h2> <p>Thanks for visting my Web Page!!</p> </div> <div class="col-lg-4"> <h2>Hi!! I am 3rd col-xs-4</h2> <p>Do visit My web page again!!</p> </div> </div> ...

As shown in the above code snippet '.col-lg-' is used. This is used to show content in columns for window size greater that 1200px.
