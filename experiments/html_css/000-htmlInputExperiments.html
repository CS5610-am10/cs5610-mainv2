1. Label and Input Tag

This experiment shows how we can link a label to a particular input field and also provide a placeholder with a default value in the input field.

Demo:

Code snippet:

... <label for="name">Name</label> <input type="text" id="name" placeholder="Ankit Masrani"/> ...

The label is linked to the input field with the for attribute. The placeholder attribute in the input field is used to provide the user information regarding the value expected in the field.

2. Datalist

This experiment shows how we can use "autocomplete" feature in html.

Demo:

Code snippet:

... <input list="cuisine" /> <datalist id="cuisine"> <option>Indian</option> <option>American</option> <option>Thai</option> <option>Mexican</option> </datalist> ...

As you can see in the above code snippet we provide a set of pre-defined options to the user. The user can select one these options by either using the input box a drop-down or through auto-complete function while typing. The user can also provide a completely new value in the input box.

3. Date and time

This experiment shows some tags introduced in HTML5 to provide date and time input functionality on an HTML page

Demo:

Code snippet:

... <input id="Date1" type="date" /> ... <input id="Datetime-local1" type="datetime-local" /> ...
