Inline CSS

This experiment shows how we can provide styling information at the tag level. However, this is not an industry standard of styling information. Professionally, we create a separate style sheet file to provide all the styling information.

Code snippet:

... <p style="background-color:maroon; color:green"> ... </p> ...

As shown in the above code snippet, we used the style attribute of the tag to modify the background color and the text color of the text.
Please note that the inline tag styling will take preference over the page level styling or any external style references.
