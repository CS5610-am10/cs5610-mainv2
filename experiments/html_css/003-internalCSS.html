Code snippet:

... <style> .test { color: green; background-color: maroon; } #testId { color: yellow; } #testId:hover { color: red; } </style> ...

We use the above style tag in the head section of this html page. '.' is used as a class selector and '#' is used an id selector. So, in our example all the tags with class 'test' will have background color as maroon. We can also describe styling information on mouse events. In the above example, the text color of an html element is altered on mouse hover.
