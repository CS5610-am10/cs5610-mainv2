External CSS

This experiment shows how we can provide styling information using an external stylesheet file. This stylesheet file can then be referenced within different web pagez and can be used to provide unified styling across the website. In this way we can modify the styling of multiple web pages by changing a single file. This manner of styling is followed within the industry.

As shown in the above code snippet, we provide all the styling information within a different stylesheet file (.css) and reference that stylesheet file within this html page.
