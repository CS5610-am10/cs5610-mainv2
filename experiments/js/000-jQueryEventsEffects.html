Mouse Events and Effects

In this experiment, I demonstrate using mouse events in jQuery to dynamically modify the html page. Please hover over the below image.

Demo:

Code snippet:

... HTML: <img id="Img1" src="../../images/batman-logo.gif" /> Javascript: $(document).ready(function () { $("#batman").hover(function () { var info = $("<h1>I am Batman!!!</h1>"); $(this).after(info.fadeIn(500)); }, function () { $(this).next().fadeOut(500); }); }); ...

As you can see from the jQuery snippet, I fetch the batman image and define the hover method for the image. The hover method takes two functions: a mouseenter() and mouseleave() methods. So, on mouseenter(), I add a header tag after the image and on mouseleave() I fetch the next sibling of the image tag (which is the header tag added in the mouseenter() event) and remove it.
