jQuery CSS Spinner

In many web pages, we see a refresh button, clicking on which reload some content from the server. In order to provide good user experience the refresh icon will rotate, conveying to the user to wait until the content is being loaded. Click on the start and stop buttons below to start and stop the spinner respectively.

Demo:

Code snippet:

... HTML: <div id="spinner-div"> <div class="row"> <div class="col-xs-1"> <button id="Button1" type="button" class="btn btn-xs btn-info">Start</button> </div> <div class="col-xs-1"> <button id="Button2" type="button" class="btn btn-xs btn-danger">Stop</button> </div> <div class="col-xs-4"> <img id="Img1" src="../../images/spinner.png" /> </div> </div> </div> Javascript: $("#spinStart").click(function () { console.log("click called"); $("#spinImage").addClass("spinner"); }); $("#spinStop").click(function () { console.log("click called"); $("#spinImage").removeClass("spinner"); }); ...

On clicking on the "start" button, the CSS class "spinner" is added to the spinner image. In the spinner class, I use CSS3 keyframes to provide the rotating effect. And on clicking on the "stop" button, the "spinner" CSS class is removed from the spinner image.
