jQuery UI Draggable

In this experiment, we see how we can use jQuery UI to create draggable UI. This particular functionality is used in various applications, especially in the teaching area where students are asked to drag and match options to a correct table heading. Here I show a small demo of matching food items to a cuisine.

Demo:

Indian food Italian food
Please drag the following food items to there respective columns in the above table.
  • Pizza
  • Butter chicken

... Each list item is made draggable through the class selector "draggable".
