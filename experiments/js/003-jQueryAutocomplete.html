jQuery Autocomplete

In this experiment, we see how we can use jQuery UI to use the autocomplete widget. I think this widget is very cool and very resourceful. We use this all the time on websites. For example: On Amazon.com when you type in "jacket" it will autocomplete to give you options: "Jackets for men" and "Jackets for women."

Demo:

Code snippet:

... HTML: <div class="ui-widget"> <label for="countries">Location: </label> <input id="countries"/> </div> Javascript: var countries = ["USA", "India", "Australia", "New Zealand"]; $("#countries").autocomplete({ source: countries }); ...

The countries list variable defined is used to assist user input. Currently we have hard-coded this list, but we can easily populate it through a server response from the database.
