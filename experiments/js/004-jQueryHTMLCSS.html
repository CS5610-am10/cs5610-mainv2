jQuery HTML and CSS

In this experiment, we see how we can use jQuery to add new elements in the html page and modify its styling based on an event. As you can see in the demo, there are 2 button: add a rectangle shape in a bordered canvas and change the color of the shapes added to the canvas. Also, the shapes added to the canvas are draggable only within the canvas border.

Demo:

Code snippet:

... HTML: <div class="row"> <div class="col-lg-2"> <button class="btn-default" id = "addRectangle">Add Rectangle</button> </div> <div class="col-lg-2"> <button class="btn-default" id = "changeBlueBtn">Blue shapes</button> </div> </div> <div id="canvas"></div> Javascript: $("#addRectangle").click(function () { console.log("add rect called"); var r = $("<canvas class='canvas-rect'/>") r.draggable({ containment: "parent" }); $(this).parent().parent().parent().find("#canvas").append(r); }); $("#changeBlueBtn").click(function () { $(".canvas-rect").css("background-color", "blue"); }); ...

The draggable movement of the rectangle is contained within the canvas by using the "containment" attribute as "parent". Also, some traversing through the DOM tree is required to add the rectangle to the div after the row div of the button. The styling of an element can be changed using the css method.
