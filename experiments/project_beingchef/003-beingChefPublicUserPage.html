BeingChef Public User Page

In the BeingChef project, a particular logged in user can follow another user with whom it may share common interests in cooking. This experiment shows how a public user page looks to any one visiting the web application.

The follow and unfollow functionality can only be excercised from the project page

The public user information is displayed as a expand collapse sections using CSS3 key frame animations with angular JS directives. This is done in order to allow the visitor of the page to easily browse through the content.
