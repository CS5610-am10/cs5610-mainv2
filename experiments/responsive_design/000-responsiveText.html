Responsive Text

This experiment is to change the behavior of text when the window size is altered. Change the window size to see below text change color and font size.

Code Snippet:

@media screen and (min-width: 800px) { p { color: red; font-size: 18px; } } @media screen and (min-width: 500px) and (max-width: 800px) { p { color: yellow; font-size: 12px; } }

As you can see from the above css snippet adjacent breakpoints are created at 500 and 800 pixels.
