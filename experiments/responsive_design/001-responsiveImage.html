Responsive Image

This experiment shows how we can make images responsive to window size. Change the window size to see the below image adapting its size to the window size. Note: Even without doing any changes in style, we can get responsive behavior. But, this experiment shows how we control the width of the image for different window sizes.

Demo:

Code Snippet:

@media screen and (min-width: 800px) { ... .responsiveImage { width: 700px; } } @media screen and (min-width: 500px) and (max-width: 800px) { ... .responsiveImage { width: 450; } } @media screen and (max-width: 500px){ .responsiveImage { width: 350px; } }

The above changes in the css file allows us to set the width of the image for various adjacent window breakpoints.
