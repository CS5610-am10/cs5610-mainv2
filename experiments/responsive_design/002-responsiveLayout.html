Responsive Layout

This experiment is to demonstrate how we can make the layout of our page responsive. The experiment demonstrates two sections which are side by side. On reducing the width of the page the layout of the page is modified to display the section one below the other.

Code Snippet:

@media screen and (min-width: 800px) { #sectionLeft { width:50%; color: red; float:left; } #sectionRight { width: 50%; color: yellow; float:left; } } @media screen and (min-width: 500px) and (max-width: 800px) { #sectionLeft { width:50%; color: red; } #sectionRight { width: 50%; color: yellow; } }

As you can see from the above css snippet adjacent breakpoints are created at 500 and 800 pixels. Explanation: The red and yellow text sections will be shown side by side for window width greater that 800px and will be shown one below the other for window width between 500px and 800px.
