Responsive Table

This experiment shows how we can achieve resposive tables. The experiment demonstrates that if the width of the window is reduced for a html page containing a table, we can transform the table into rows containing "heading-value" pairs.
Go ahead and change the width of the window.

Demo:

First Name Last Name
Ankit Masrani
Jose Annunziato

Code Snippet:

... @media screen and (min-width: 500px) and (max-width: 800px) { table, thead, tbody, th, td, tr { display: block; } thead tr { display: none; } td:nth-of-type(1):before { content: "First Name"; padding-right: 10px; } td:nth-of-type(2):before { content: "Last Name"; padding-right: 10px; } tr:nth-of-type(even) { background-color: lightblue; } tr:nth-of-type(odd) { background-color: blue; } } ...

Explanation: We modify the manner in which the table is rendered in the 500px and 800px window segment.

  1. The table components are rendered as block elements.
  2. The table head is not displayed
  3. td:nth-of-type(1):before This means that before every first td element within a parent tag apply some styling. In our case, we add the table head contents that were removed from display.
  4. tr:nth-of-type(even) This lets us define styling for every even row.
