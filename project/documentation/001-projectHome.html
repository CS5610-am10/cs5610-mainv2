BeingChef Home and Navigation Bar

Shown above is the BeingChef landing page.

Usecases on this page:

Search:
Any user can search for recipes provided by the BigOven API
Login/Register:
The user can login or register using the 'Login' and 'Register' buttons in the navigation bar respectively. Essentially, the user can login/register from any particular page within application. Also, the user cannot bookmark/add to favorite a recipe, add a review or rating for a recipe or add recipe ingredients to grocery list with logging or registering into the system.
Go to Home:
The user can navigate back to this page from anywhere within the application by clicking on the BeingChef icon on the top left of the page.
View About/Contact:
By clicking on the 'about' and 'contact' links in the footer of the page the user can view the information about the application or view contact information about the developer.
