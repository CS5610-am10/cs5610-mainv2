BeingChef Search

  • Searching for recipes will navigate the user to the page shown below. The search string is 'pizza'. The food item recipes are fetched from the BigOven API (api.bigoven.com). Following features are visible on the page:

    1. Number of results fetched
    2. Time taken to fetch the results
    3. The page displays 10 results per page with a facility to navigate to other search results
    4. The user can view the dish name and an image of the dish on this page. Details about the food item can be viewed by clicking on the dish name link.
    5. The user can also search more recipes from this page too instead of navigating back to the home page.
    6. If the dish text did not return any results, the user is shown an appropriate error message.
