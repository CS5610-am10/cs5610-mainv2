BeingChef Recipe Details

  • By clicking on a particular recipe in the search results the user is navigated to this page.

    Following dish information can be viewed on this page:

    1. Name
    2. Image
    3. Description
    4. Cuisine
    5. Ingredients
    6. Instructions
    7. Servings

    After being logged into the system the user can either mark the dish as a favorite, add a review for the dish or add the dish ingredients to his grocery list.
