BeingChef Responsiveness

Application responsiveness has become integral part of design in today's world with varying device screen sizes. More and more users using their mobile phones to view web applications. Although BeingChef is best viewed on a screen size of 1200 X 500, it is designed to handle smaller screen sizes too. Most of the responsiveness is handled by various Bootstrap classes, images in the search results and user dashboards are handled using CSS "media" queries.
