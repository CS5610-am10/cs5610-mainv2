Future Scope and Improvements

This application has a lot potential for future development. Some of the features, I feel that are important are:

  1. Users can add recipes
  2. Tagging recipes like quick bites or involving a particular main ingredient
  3. Providing advanced search features, for example allowing the user to search by cook time
  4. Developing a special mobile version of the application to achieve better ease of use
